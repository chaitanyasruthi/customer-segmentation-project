# Customer Segmentation Project

This project uses **Python** and **K-Means Clustering** to group customers based on their spending behavior.

## Features
- Customer segmentation
- Data analysis
- Data visualization using Matplotlib

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Output
Customers are divided into different clusters based on:
- Annual Income
- Spending Score

## Conclusion
This project helps businesses understand customer behavior and improve marketing strategies.
